Demo video --- A smart home monitoring system for use with the Phidget hardware platform. This project consisted of two parts, Java server with integrated web server and lightweight web client.

Server:
  • Multithreaded Java application server
  • Multiple connects to hardware Phidgets over network or USB
  • Read sensor data in real time and determine events from this stream
  • User defined events using JSON
  • High level entity detection event rule
  • Inbuilt web server for serving of simple web API with SSL and user authentication
  • Server push support for event feed

Client:
  • Connects to server and utilise API
  • Display event feed from server
  • Display real time sensor values using Charts.js
  • Use of AJAX for data retrieval
